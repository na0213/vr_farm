# 🐄 FARM360

全国の牧場を360°画像で体験できるWebプラットフォームです。  
生産者の想いや飼育環境を可視化し、消費者との距離を縮めることを目的としています。

🔗 https://www.farm360.jp/

---

## 🌍 コンセプト

「牧場の“空気”を届ける。」

テキスト情報だけでなく、360°体験を通じて、
生産現場の透明性を高め、動物福祉への理解を促進することを目指しています。

---

## 🚀 技術スタック

### Backend
- Laravel
- MySQL
- REST風ルーティング設計

### Frontend
- Blade
- Tailwind CSS
- Alpine.js
- Vite

### Infrastructure
- Heroku（デプロイ）
- JawsDB（MySQL）
- AWS S3（画像保存）

---

## ✨ 主な機能

### 👤 ユーザー機能
- 牧場一覧表示
- 360°VR画像閲覧
- 詳細ページ閲覧

### 🛠 管理者機能
- ログイン認証
- 牧場情報のCRUD
- 360°画像アップロード

---

## 📸 360°表示

Ricoh Thetaで撮影した360°画像を使用し、
ブラウザ上で没入型体験が可能です。
